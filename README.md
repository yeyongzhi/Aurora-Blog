# 📦️ shadcnUI-vue-ts-template

## 🚀项目介绍
一个基于ShadcnUI 的 简单 `vue3 + typescript + vite` 项目模板
开箱即可使用
当然这仅仅是一个方便你安装这几个依赖的模板（脚手架），你可以根据自己的需求进行修改
然后开始你的项目

pnpm dlx shadcn-vue@latest add button
